Get model and data from model fitted object

Usage

model.info(fit, ...)

Arguments

fit
a model fitted object (see as.unit.tree for details)
...
additional arguments to be passed to model.info

Get model and data from model fitted object

Value

a classed object compatibile with a make.model.phylo function.

Description

Generic function for pulling apart model fitted objects and obtaining phylogeny, data, model type and parameters

Details

These functions are primarily used internally within the function as.unit.tree. A different model.info.xx function is built for every implemented class of model fitted object and is tailored to the specifics of the form of model output. The return value will differ depending on the model used. For example, if a single phylogeny is supplied which has been used to fit a single rate model, model.info will return a object of class fitC, consisting of the original tree to be rescaled, the original comparative dataset, the model class (e.g. "BM", "OU", etc.) and the model-specific parameters.

The returned object needs to be of a class for which a make.model.phylo function exists. make.model.phylo will take the object output by model.info and return a 'unit.tree' object.

Examples

## finch data
data(finch)
phy <- finch$phy
data <- finch$data[,"wingL"]
