REML estimate of BM parameter

Usage

sigsq.est(unit.tree)

Arguments

unit.tree
a unit.tree object

REML estimate of BM parameter

Value

m.pic estimate of sigma2

Description

Calculates the mean of the squared contrasts, which is equal to the REML estimate of the BM rate parameter. Used as a summary statistic to evaluate model adequacy

Details

This summary statistic is used to assess the ability of the model to adequate estimate the rate of evolution. On datasets simulated on a unit.tree (see sim.char.unit), the expected value is equal to 1.

The REML estimate of sigma2 is included as a default summary statistic in the function summ.stats

Examples

data(finch)
phy <- finch$phy
dat <- finch$data[,"wingL"]
unit.tree <- as.unit.tree(phy, data=dat)

## Estimate sigsq from unit.tree

sigsq.est(unit.tree)
