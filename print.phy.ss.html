Print summary statistics

Usage

"print"(x, ...)

Arguments

x
a phy.ss object
...
additional arguments to be passed to print

Print summary statistics

Value

prints p.values to screen

Description

Generic function for printing 'phy.ss' objects from compare.summ.stats

Examples

data(finch)
phy <- finch$phy
dat <- finch$data[,"wingL"]

## use phy.model.check to assess model adequacy
## pretend as if tree has already been rescaled
p <- phy.model.check(phy, data=dat, nsim=10)

## print output
p
