Simulate character evolution on unit tree

Usage

sim.char.unit(unit.tree, nsim = 1000)

Arguments

unit.tree
a 'unit.tree' object
nsim
the number of datasets to simulate (if single unit.tree is provided; see below)

Simulate character evolution on unit tree

Value

A list of unit.trees. In each unit.tree, the phylogeny will be the same as that of the input unit.tree(s) but the data and the contrasts will be replaced by the simulated datasets.

Description

Simulates continuous characters along a unit tree according to a Brownian motion process with rate 1.

Details

If a single unit.tree is supplied, the function will simulate nsim datasets under a Brownian motion process with a rate of 1. If a lits of unit.trees supplied (such as those derived from a Bayesian analysis) sim.char.unit will simulate a single data set on each tree in the list and the nsim argument will be ignored.

Examples

data(finch)
phy <- finch$phy
dat <- finch$data[,"wingL"]

## build unit.tree object
unit.tree <- as.unit.tree(phy, data=dat)

## simulate 2 datasets on unit tree
sims <- sim.char.unit(unit.tree, nsim=2)

sims
