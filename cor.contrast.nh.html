Relationship between contrasts and node height

Usage

cor.contrast.nh(unit.tree)

Arguments

unit.tree
a unit.tree object

Relationship between contrasts and node height

Value

s.hgt the estimated slope parameter

Description

Fits a linear model between the absolute value of the contrasts and the node height at which they were calculated

Details

This summary statistic is the estimate of the slope from fitting a linear model between the contrasts and the node height (i.e. tree depth) at which they were calculated. It is used to evaluate whether the model is adequately capturing variation relative to tree depth. If the model is adequate, we expect there not to be a relationship between the contrasts and their node height (i.e. slope ~ 0). Note that the node heights being used here are the absolute node ages from the original phylogeny and not the adjusted node ages obtained from computing contrasts (see pic).

The slope of the contrasts and their variances is included as a default summary statistic in the function summ.stats

Examples

data(finch)
phy <- finch$phy
dat <- finch$data[,"wingL"]
unit.tree <- as.unit.tree(phy, data=dat)

## estimate slope
cor.contrast.nh(unit.tree)



[1] -0.306688
