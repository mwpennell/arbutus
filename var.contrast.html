Variance of contrasts

Usage

var.contrast(unit.tree)

Arguments

unit.tree
a unit.tree object

Variance of contrasts

Value

the estimate of the variance of the absolute value of the contrasts

Description

Calculates the variance of the absolute value of the contrasts

Details

This summary is used to evaluate whether the model is sufficiently capturing the variation in rates across the tree.

The variance of contrasts is NOT included as a default summary statistic in the function summ.stats

Examples

data(finch)
phy <- finch$phy
dat <- finch$data[,"wingL"]
unit.tree <- as.unit.tree(phy, data=dat)

## estimate variance of contrasts
var.contrast(unit.tree)
